Critical Change: H₂ Recycling Loop (Hydrogenotrophic Methanogenesis)

In your current model, H₂ is being separated out and wasted. Instead, it must be recycled back into the methanogenesis reactor. This reaction converts CO₂ + H₂ directly into methane:


CO₂ + 4H₂ → CH₄ + 2H₂O     (ΔG° = −130.7 kJ/mol)


This single change alone can reduce CO₂ by 15–25% and increase CH₄ yield proportionally.

4
Change DIGEST1 to thermophilic: Set temperature = 55°C (328 K). Add stoichiometric reactions R1–R5 from Section 5. Set pH controller to maintain 5.5–6.0. Remove any O₂ from inputs.
5
Add H₂ recycle stream to DIGEST2: In the PFD, draw a stream from the gas outlet of DIGEST1 back to the inlet of DIGEST2. This routes H₂ gas to react with CO₂ via reaction R7 (CO₂ + 4H₂ → CH₄ + 2H₂O). Set fractional conversion = 0.90 for this reaction.
6
Increase DIGEST2 HRT to 15 days: Calculate reactor volume = total inlet flow (kg/hr) × HRT (hr) / density (kg/m³). Add reactions R6–R10 from Section 5. Set temperature = 55°C, pressure = 1.5 atm, pH = 7.2.
7
Add dedicated H₂S removal block BEFORE scrubber: Insert a Sep2 block between B2 and SCRUBBER. Set H₂S component split fraction = 0.99 to waste stream. NH₃ split fraction = 0.70 to waste stream.
8
Fix SCRUBBER pressure: Add a COMP block before SCRUBBER to pressurize gas to 9 bar. In SCRUBBER, set pressure = 9 bar, temperature = 12°C. Use RadFrac with 10 stages instead of Sep2.
9
Fix PSA block: In the Sep2 PSA block, set split fractions: CH₄ → product stream = 0.97; CO₂ → waste = 0.98; H₂S → waste = 1.0. Operating pressure = 7 bar. Add COOLER2 at 10°C to condense water before product storage.
10
Run sensitivity analysis: Use Aspen's sensitivity tool to vary DIGEST2 temperature (45–60°C) and HRT (10–20 days) and plot CH₄ mole fraction in the final product. Optimal is ~55°C and 15 days per Aspen Plus literature.
